# Django CRUD Module

ငွေသုံးစာရင်းကို Django နဲ့ ရေးမယ်။

## Features

- [ ] ကိုယ့်ရဲ့ လက်ကျန်ငွေ ကြည့်လို့ရမယ်။ (Balance)
- [ ] ဝင်ငွေ ထွက်ငွေ အမျိုးအစား စာရင်း (Transaction Category)
- [ ] ဝင်ငွေ ထွက်ငွေ မှတ်တမ်းများ
